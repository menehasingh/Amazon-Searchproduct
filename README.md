# Amazon-Searchproduct
This project is a Selenium automation framework developed using Java, TestNG, and Maven.
It automates the Amazon search functionality and validates product results.
## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports
- ## Project Structure
src/test/java
 ┣ base
 ┣ page
 ┣ test
 ┣ utils
 ┗ listener
 ## Features
- Page Object Model (POM)
- Extent Report integration
- Screenshot on failure
- TestNG listeners
- ## How to Run
### Run using Maven:
mvn clean test
### Run using TestNG:
Right click on testng.xml → Run as TestNG Suite
